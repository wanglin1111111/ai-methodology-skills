# LingGuang AI Product Strategy Skill

## 技能描述
管理"灵光"AI产品策略知识库，支持策略查询、决策追踪、假设验证、竞争分析等功能。用于产品规划、市场分析、竞品对比等场景。

**隐私保护声明**：
- 本技能仅处理公开的产品策略信息
- 不存储、传输或暴露任何个人隐私数据
- 所有知识库内容均为公开可分享的产品分析
- 不涉及用户个人信息、账号数据或敏感业务数据

## 触发条件
- 用户询问"灵光"产品相关问题时
- 需要查询产品决策记录时
- 需要验证产品假设时
- 需要分析竞争态势时
- 用户提到"AI交互"、"工作流引擎"、"私域沉淀"、"Agent个性化"等关键词

## 知识库位置
`lingguang-strategy/knowledge-base/lingguang-ai-product-strategy.md`

## 核心功能

### 1. 策略查询
```javascript
// 查询传播策略
queryStrategy('传播策略')

// 查询产品定位
queryStrategy('产品定位')

// 查询工作流理念
queryStrategy('工作流引擎')
```

### 2. 决策追踪
记录产品关键决策：
- 决策点
- 选择方案
- 决策理由
- 后续影响

### 3. 假设验证
跟踪待验证假设的验证进度：
- 假设内容
- 验证方法
- 验证结果
- 后续行动

### 4. 竞争分析
对比分析竞争优势：
- 交互创新
- 生态定位
- 技术架构
- 商业模式

## 灵光产品策略摘要

- 目标用户：生活创作者

### 关键决策

- 工作流引擎（交付为本）
