## 🔬 核心概念

### 1. AIC（AI Center/Community）

**定义**：人工智能中心/社区，专注于AI安全研究与实践

**核心特色**：
- 三方语料库（东方/西方/综合）
- Ideas分享而非单向输出
- 东西方协作模式

### 2. Skon/Skill提升计划

```yaml
Skon（Strategic Knowledge）: 战略知识提升
  - 职业规划
  - 战略思维
  - 行业洞察

Skill: 技能提升
  - 硬技能：AI安全技术
  - 软技能：沟通、协作、领导力
```

### 3. 资金支持体系

```yaml
支持类型:
  研究资助: 实质性投入（非数千级别）
  奖学金: 学费支持 + 生活补贴
  项目奖金: 根据成果影响力评估

申请流程:
  提交研究计划 → 导师推荐 → 审批委员会 → 拨款执行
```

### 4. 超级活跃区（Super Active Area）

```yaml
特征:
  讨论密度: 每日数十个话题
  快速响应: 平均响应时间<30分钟
  深度交流: 问答 + 思想碰撞

参与形式:
  - 问题回答
  - 聊天互动
  - 项目协作
  - 活动组织
```
