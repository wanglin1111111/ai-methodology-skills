---

## 🎯 核心理念

### 理念创新

\\\yaml
传统黑客松: 软件开发竞赛 → 技术产出
长寿月黑客松: 迷你营地活动 → 健康思维培养

核心理念:
  - 打破常规
  - 关注健康和长寿
  - 建立健康思维模式
  - 非单纯技术产出
\\\

## 📋 项目结构四要素

\\\yaml
1. 提出问题: 明确想要解决的挑战或疑问
2. 制定计划: 设计具体的实验方案
3. 提供解决方案: 基于问题和计划给出解决办法
4. 成果展示: 符合长寿月的整体结构要求
\\\

---

## 🏆 评审机制

\\\yaml
传统评审: 专家打分制
创新评审: 公众投票

打卡标准:
  - 问题的趣味性
  - 方法的创新性
  - 其他多维度标准

评审者: 所有参与长寿月的人员
\\\
