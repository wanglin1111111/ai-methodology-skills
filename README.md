## 技术架构

### 多模态交互
```
提示词（文本） ──┐
                 │
音乐（音频） ────┼──→ AI模型 ──→ 视觉输出
                 │
手势（视觉） ───┘
```

### 核心技术
- **AI实时生成**: 提示词→代码→视觉
- **音频分析**: 音乐→节奏参数→视觉调制
- **手势识别**: 摄像头→手部位置→交互控制
