## 核心概念

### "命运"二元拆解
- **命**: 静态配置（车型配置）
- **运**: 动态轨迹（行驶路况）

### "山医命相卜"五大板块
- **山**: 道家哲学、修炼养生
- **医**: 中医养生、健康管理
- **命**: 占卜、八字、命运预测
- **相**: 风水、面相、环境分析
- **卜**: 推演、预测、决策辅助

### 品牌哲学
- **Face Telling**: 像展开书籍一样呈现命运
- **墨具五色**: 水墨看似单一，实则包含五色
- **天人合一**: 人与自然宇宙和谐统一

### 数据策略
- **死数据**: 古籍文献（固定不变）
- **活数据**: 用户交互（动态更新）
