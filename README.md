## 核心概念

### 音乐基因（Music Genes）

将音乐拆解为最小元素，而非按流派分类：

| 基因类型 | 示例 | 特征 |
|---------|------|------|
| **节奏基因** | 四拍底鼓、跳跃贝斯、快速踩镲 | 控制节拍、律动 |
| **旋律基因** | 主旋律线条、和声进行 | 控制音高、走向 |
| **情感基因** | 情感铺底、激情爆发、忧郁氛围 | 控制氛围、情感 |
| **音色基因** | 合成器音色、效果器 | 控制声音质感 |
| **结构基因** | 主歌、副歌、桥段 | 控制曲式结构 |

### 代码化创作优势

1. **透明可控**: 符号化生成过程透明
2. **易于修改**: 参数调整立即见效
3. **便于协作**: 代码可版本控制
4. **可理解性**: 学习创作过程原理

### 游戏化应用

#### 泡泡游戏
- 音乐基因可视化
- 拖拽组合交互
- 实时试听调整

#### 老虎机机制
- 随机基因组合
- 惊喜探索体验
- 鼓励意外创新

#### 俱乐部对战
- DJ台上对战
- 击败对手获取基因
- 处理基因冲突
- 策略性组合

## 核心批判

### 对现有工具的问题

1. **DAW界面复杂**: 如"飞船控制台"，学习成本高
2. **软件封闭**: 非开源，难以与AI协作
3. **订阅费用**: Auto-Tune等专业插件经济负担重
4. **代码语言门槛**: Sonic Pi等需要编程基础

### 对AI生成的质疑

1. **疏离感**: 朋友忘记自己用Suno生成的歌
2. **缺乏参与**: 创作者与作品脱节
3. **情感连接弱**: 纯结果导向失去创作乐趣
