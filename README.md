# Skills Repository

这是一个OpenClaw技能仓库，包含多个实用技能包。

## 技能列表

### 1. 内容创作技能包
营销文案、内容创作辅助工具。

### 2. 写作辅助技能包
语法规则、修辞指南、风格指南、引用格式。

### 3. 灵光AI产品策略技能包 ⭐ NEW
灵光AI产品策略知识管理技能，支持策略查询、决策追踪、假设验证、竞争分析。

**功能特点**：
- 策略查询（传播策略、产品定位、工作流引擎）
- 决策追踪（关键决策记录与效果追踪）
- 假设验证（验证进度管理）
- 竞争分析（竞品对比与市场定位）

**隐私保护**：
- 仅处理公开的产品策略信息
- 不存储个人隐私数据
- 知识库内容可公开分享

**文件位置**：`lingguang-strategy/`

## 安装方法

将需要的技能包复制到OpenClaw技能目录：
```
~/.openclaw-autoclaw/skills/[技能包名称]/
```
