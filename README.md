## 🚀 完整工作流

### 核心环节

| 环节 | 功能 | 关键点 |
|------|------|--------|
| **视频生成** | 核心内容生成 | 高质量+快速迭代 |
| **音频同步** | 音频与视频同步 | 时间同步+情绪匹配 |
| **字幕生成** | 自动生成字幕 | 准确性+时间对齐 |
| **角色一致性** | 确保角色一致 | Character and Custom |
| **后期处理** | 最终整合优化 | 整体协调+质量提升 |

## 🎯 Character and Custom技能

### 核心能力

```yaml
能力1: 多角度3D视图生成
  - 正面、侧面、背面视图
  - 其他角度视图

能力2: 细节捕捉
  - 材质细节
  - 颜色信息
  - 纹理特征

能力3: 提示词约束
  - 约束后续视频生成
  - 确保角色一致性
```

```yaml
参考类型:
```

```yaml
步骤6: 后期处理（15分钟）
```
