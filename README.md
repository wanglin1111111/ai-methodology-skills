## 安装方法

### 方法1: 直接复制
将此目录中的文件复制到OpenClaw技能目录：
```bash
cp -r lingguang-skill-repo ~/.openclaw-autoclaw/skills/lingguang-strategy/
```

### 方法2: Git克隆（推荐）
```bash
cd ~/.openclaw-autoclaw/skills/
git clone https://github.com/wanglin1111111/-.git lingguang-strategy
```

## 使用方法

### 在OpenClaw中使用

1. **查询策略**
```
用户：灵光的传播策略是什么？
助手：[自动读取知识库] 灵光传播策略核心是...
```

2. **记录决策**
```
用户：记录决策：选择支付宝支付
助手：[更新知识库] 已记录...
```

3. **验证假设**
```
用户：验证假设：用户愿意为AI交互体验付费
助手：[标记验证中] 已标记...
```

## 核心功能

### 1. 策略查询
- 传播策略与私域沉淀
- 产品定位与未来展望
- 硬件产品功能
- AI工作流引擎理念
- Agent个性化

### 2. 决策追踪
- 记录产品关键决策
- 追踪决策效果
- 定期复盘

### 3. 假设验证
- 标记待验证假设
- 记录验证进度
- 更新验证结果

### 4. 竞争分析
- 对比竞品优势
- 分析市场定位
- 监控行业动态
