---

## 📖 概述

本技能专注于Roost非营利组织及其开源安全工具生态，介绍三大核心项目的功能与应用场景。

**核心项目**：
- **Osprey**: 实时事件检测引擎
- **Coop**: 内容审核与审查工具
- **Groce**: 联合模型（文本分类）

---

## 🔬 核心概念

### Roost使命

```yaml
性质: 美国501(c)(3)非营利组织
使命: 成为安全领域的Apache/Linux
理念:
  - 开源优先
  - 数据隐私保护
  - 社区驱动
许可: Apache 2.0（允许商业化）
```

### 信任与安全四阶段

| 阶段 | 目标 | 措施 |
|------|------|------|
| **保护** | 预防有害行为 | 实时检测、自动过滤 |
| **调查** | 查明行为详情 | 内容审核、证据收集 |
| **审查** | 评估严重性 | 人工审核、策略应用 |
| **执行** | 采取处理措施 | 内容删除、账号处罚 |

## 🚀 三大核心项目

### Osprey（检测引擎）

```yaml
定位: 实时事件检测引擎
来源: Discord内部工具捐赠
应用: Blue Sky、Discord生产环境

核心能力:
  - 账号劫持检测
  - 非法内容检测
  - AI代理输出识别
  - 聚类分析（新型威胁）
```

### Coop（审查工具）

```yaml
定位: 内容审核与审查工具
许可: Apache 2.0（Roost收购后开源）

核心能力:
  - 大规模人工审核
  - 自动化辅助
  - 高精度数据标注
  - 自定义模型集成

内置检测: 儿童虐待材料、非法内容
用户案例: Notion等平台
```

### Groce（联合模型）

```yaml
定位: 联合模型（文本分类）
合作方: AI实验室

核心特性:
  - 导入自有策略
  - 自定义文本分类器
  - 提供解释性理由
  - 权重公开可用
```

| **文本分类** | Groce | 定制化、可解释性 |
| **综合方案** | 三者组合 | 全流程覆盖 |
