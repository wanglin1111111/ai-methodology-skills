## 🔬 核心认知

### 大模型定位

```yaml
观点: 模型不仅是工具
定位: 类似操作系统的"内核"（Kernel）

架构类比:
  Kernel: 大模型（核心能力）
  Harness: 应用层/驱动（应用场景）
  
完整生态: Kernel + Harness
```

### 知识民主化

```yaml
互联网时代: democratize信息获取
AI时代: democratize知识获取

演进: 信息 → 知识
意义: 获取门槛持续降低
```

### 模型尖峰特性

```yaml
特性: 某些领域极强，其他领域可能较弱
策略: 接受不完美性，利用优势领域
```

