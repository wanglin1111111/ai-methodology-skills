## 使用技巧

### 1. 组合使用多个技能

不要孤立使用单个技能，要组合应用：

```
创业 = 动机 + 挑战 + 机会 + 产品 + 品牌
个人品牌 = 独特性 + 产品化 + IP + 社区
技术项目 = LLM + 评估 + 架构 + 设计
```

### 2. 制定个人学习计划

```markdown
# 我的AI学习计划

## 本月目标
- [ ] 完成3个技能学习
- [ ] 实践1个项目

## 具体安排
| 周次 | 技能 | 实践 |
|------|------|------|
| 第1周 | ai-era-human-value | 写反思笔记 |
| 第2周 | content-creation-core | 创作3篇内容 |
| 第3周 | ai-marketing-efficiency | 优化营销方案 |

## 复盘
- 学到了什么？
- 应用效果如何？
- 下一步计划？
```

### 3. 建立学习小组

- 找到志同道合的学习伙伴
- 定期分享学习心得
- 互相监督和鼓励

### 4. 持续迭代

- 每季度回顾学习成果
- 根据反馈调整计划
- 不断更新知识体系
