## 核心理念

### 迭代公式

```
迭代开发 = 小步快跑 × 快速验证 × 持续改进
```

### 迭代循环

```
规划 → 开发 → 测试 → 发布 → 反馈 → 改进
```

## 方法论框架

### 迭代开发三步法

**第一步：规划**
- 确定迭代目标
- 设计最小功能
- 制定迭代计划

**第二步：执行**
- 快速开发实现
- 及时测试验证
- 快速发布上线

**第三步：改进**
- 收集用户反馈
- 分析改进方向
- 进入下一迭代
