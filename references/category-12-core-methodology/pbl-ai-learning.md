## 核心理念

### PBL公式

```
PBL学习 = 真实项目 × 问题导向 × 实践探索 × 反思总结
```

### 学习循环

```
项目 → 问题 → 探索 → 实践 → 反思 → 提升
```

## 方法论框架

### PBL学习三步法

**第一步：项目**
- 选择真实项目
- 明确学习目标
- 设计学习任务

**第二步：实践**
- 动手实践探索
- 解决实际问题
- 积累实践经验

**第三步：反思**
- 总结学习收获
- 反思改进方向
- 进入新项目

| 误区 | 正确认知 |
