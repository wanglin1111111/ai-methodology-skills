## 核心理念

### 赛道选择五维模型

```
┌──────────────────────────────────────────┐
│  维度1：市场规模                           │
│  → 市场空间有多大？                         │
│  → 是存量市场还是增量市场？                 │
├──────────────────────────────────────────┤
│  维度2：竞争格局                           │
│  → 有哪些竞争者？                           │
│  → 差异化空间有多大？                       │
├──────────────────────────────────────────┤
│  维度3：技术成熟度                         │
│  → 技术是否成熟？                           │
│  → 是否到了商业化时点？                     │
├──────────────────────────────────────────┤
│  维度4：团队匹配度                         │
│  → 团队是否擅长这个领域？                   │
│  → 是否有相关资源？                         │
├──────────────────────────────────────────┤
│  维度5：可持续性                           │
│  → 赛道是否有长期价值？                     │
│  → 是否会很快被淘汰？                       │
└──────────────────────────────────────────┘
```

### 赛道评估公式

```
赛道得分 = 市场规模 × 竞争友好度 × 技术成熟度 × 团队匹配 × 可持续性
```

## 方法论框架

### 赛道选择四步法

**第一步：市场分析**
- 评估市场规模和增速
- 分析市场阶段（蓝海/红海）
- 识别市场痛点

**第二步：竞争分析**
- 识别主要竞争者
- 分析竞争格局
- 寻找差异化空间

**第三步：技术评估**
- 评估技术成熟度
- 判断商业化时点
- 识别技术风险

**第四步：匹配评估**
- 评估团队能力匹配
- 评估资源匹配
- 综合打分排序

## 常见误区警示

| 误区 | 正确认知 |
|------|---------|
| ❌ "选热门赛道一定对" | ✅ "热门赛道竞争激烈，可能不是好选择" |
| ❌ "市场规模越大越好" | ✅ "适合团队的赛道才是好赛道" |
| ❌ "技术越新越好" | ✅ "技术成熟且商业化时机对才好" |
| ❌ "赛道选择后就固定" | ✅ "可以随着认知深入调整赛道" |
