## 核心理念

### 职业身份的新定义

传统定义：职业 = 职位 + 公司 + 行业
AI时代定义：职业 = 解决的问题 + 创造的价值 + 独特的方法

### 身份重塑公式

```
新职业身份 = 旧能力 × 新场景 × 差异化价值
```

## 方法论框架

### 身份重塑三步法

**第一步：解构旧身份**
- 列出当前身份的所有要素
- 识别哪些是"外壳"（职位、公司）
- 识别哪些是"内核"（能力、价值）

**第二步：重构新身份**
- 基于内核能力，寻找新的应用场景
- 设计新的价值主张
- 创建新的身份标签

**第三步：传播新身份**
- 更新个人品牌材料
- 通过内容输出建立新形象
- 寻找新身份的验证机会

## 参考案例

**案例**：从HR到"组织文化设计师"

**背景**：传统HR经理，担心被AI替代

**解构**：
- 外壳：HR经理、招聘、薪酬
- 内核：理解人性、设计流程、文化塑造

**重构**：
- 新身份："组织文化设计师"
- 新价值：用设计思维重塑企业文化
- 新标签：帮助AI时代企业保持人文温度

**结果**：成功转型为组织文化顾问，收入翻倍。
