# 生命参数丰富化方法

> 拓展生命参数，避免生命单薄

## 🎯 快速参考卡

| 项目 | 内容 |
|------|------|
| **触发词** | "生活单调"、"生命单薄"、"丰富生活"、"拓展体验" |
| **核心观点** | 生命丰富性来自多元体验 |
| **输出目标** | 帮助用户拓展生命参数 |
| **相关技能** | personal-uniqueness-building, self-as-product-methodology |

## 核心理念

### 生命参数模型

```
生命丰富度 = 体验广度 × 认知深度 × 关系密度
```

### 生命参数维度

| 维度 | 示例 | 丰富化方法 |
|------|------|------------|
| **体验** | 旅行、学习、尝试 | 主动尝试新事物 |
| **认知** | 阅读、思考、交流 | 跨领域学习 |
| **关系** | 朋友、导师、社群 | 建立多元连接 |

## 方法论框架

### 生命丰富化四步法

**第一步：盘点现状**
- 列出当前生命参数
- 识别单薄的维度
- 找到拓展方向

**第二步：设计拓展**
- 选择要拓展的维度
- 设定具体目标
- 制定行动计划

**第三步：实践体验**
- 执行拓展计划
- 记录体验感受
- 积累新参数

**第四步：整合内化**
- 反思体验收获
- 整合到自我认知
- 形成新的生命参数
