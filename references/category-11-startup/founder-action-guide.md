## 核心理念

### 创始人时间分配

```
┌──────────────────────────────────────────┐
│  战略层（20%）                           │
│  → 定方向、做决策、找资源                 │
├──────────────────────────────────────────┤
│  执行层（30%）                           │
│  → 关键任务、重要会议、客户沟通           │
├──────────────────────────────────────────┤
│  管理层（30%）                           │
│  → 团队管理、流程优化、文化建设           │
├──────────────────────────────────────────┤
│  学习层（20%）                           │
│  → 学习提升、拓展认知、建立人脉           │
└──────────────────────────────────────────┘
```

### 创始人每日三件事

| 优先级 | 事项 | 标准 |
|--------|------|------|
| **第一** | 最重要的一件事 | 不做公司就死 |
| **第二** | 重要的一件事 | 影响公司长期发展 |
| **第三** | 必要的一件事 | 维持公司运转 |

## 方法论框架

### 创始人每日行动清单

**早晨（8:00-10:00）**
- [ ] 处理最重要的一件事
- [ ] 回复关键消息
- [ ] 准备当日重点

**上午（10:00-12:00）**
- [ ] 关键会议/决策
- [ ] 客户/投资人沟通
- [ ] 重要任务推进

**中午（12:00-14:00）**
- [ ] 午餐+休息
- [ ] 非正式交流

**下午（14:00-18:00）**
- [ ] 团队管理
- [ ] 流程优化
- [ ] 处理突发事件

**晚上（18:00-20:00）**
- [ ] 复盘当日
- [ ] 规划明日
- [ ] 学习提升

## 行动检查清单

- [ ] 每天确定最重要的三件事
- [ ] 优先完成最重要的一件事
- [ ] 减少低价值会议
- [ ] 保护深度工作时间
- [ ] 定期与团队一对一沟通
- [ ] 每周复盘和规划
- [ ] 持续学习和提升
- [ ] 建立支持系统（导师、同行）

## 常见误区警示

| 误区 | 正确认知 |
|------|---------|
| ❌ "创始人要做所有事" | ✅ "创始人要做最重要的事，其他委托他人" |
| ❌ "忙碌就是高效" | ✅ "聚焦高价值活动才是高效" |
| ❌ "会议越多越好" | ✅ "减少会议，增加深度工作时间" |
| ❌ " micromanagement 是负责" | ✅ "授权和信任是更好的管理" |

## 实用对话模板

**用户**："创始人每天应该做什么？"

**AI**：
> "创始人应该聚焦三件事：
> 
> **第一件事：最重要的事**
> - 如果不做，公司就死
> - 通常是融资、关键客户、产品决策
> - 每天优先完成这件事
> 
> **第二件事：重要的事**
> - 影响公司长期发展
> - 通常是战略、团队、文化
> - 每天投入一定时间
> 
> **第三件事：必要的事**
> - 维持公司运转
> - 通常是日常管理、流程
> - 尽量授权他人
> 
> 你今天最重要的一件事是什么？"

## 参考案例

**案例**：高效创始人的一天

**背景**：连续创业者，管理3家公司

**日程**：
- 8:00-10:00：处理最重要的事（不查看消息）
- 10:00-12:00：关键会议（控制在3个以内）
- 14:00-16:00：团队管理（一对一沟通）
- 16:00-18:00：学习提升（阅读、思考）
- 20:00-21:00：复盘和规划

**原则**：
- 每天只安排3个会议
- 每天2小时深度工作
- 每周一天"无会议日"

---

**核心结论**：创始人的核心任务是"做最重要的事，其他委托他人，保护深度工作时间"。

## 核心结论

> **AI时代，人的核心价值不是'会做什么'，而是'知道自己想要什么'。**

需求表达能力将成为新的稀缺资源。投资于明确自我、表达需求、构建独特世界，是AI时代最具价值的能力建设。
