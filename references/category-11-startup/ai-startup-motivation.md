## 核心理念

### 创业的三种动机

```
┌─────────────────────────────────────────────────────────┐
│  第一层：逃避型动机                                       │
│  → 不想上班                                               │
│  → 讨厌现在的老板                                         │
│  → 觉得创业很酷                                           │
│  ⚠️ 这种动机失败率>90%                                    │
├─────────────────────────────────────────────────────────┤
│  第二层：机会型动机                                       │
│  → 看到市场机会                                           │
│  → 有技术/资源优势                                        │
│  → 想赚大钱                                               │
│  💡 这种动机成功率~50%                                    │
├─────────────────────────────────────────────────────────┤
│  第三层：使命型动机                                       │
│  → 强烈想解决某个问题                                     │
│  → 不解决就浑身难受                                       │
│  → 愿意为此承受长期痛苦                                   │
│  🌟 这种动机成功率最高                                    │
└─────────────────────────────────────────────────────────┘
```

### 初心检验三问

1. **如果知道会失败，你还愿意做吗？**
   - 逃避型：不会
   - 机会型：可能不会
   - 使命型：依然会尝试

2. **如果不需要钱，你还会做吗？**
   - 逃避型：不会
   - 机会型：可能不会
   - 使命型：依然会做

3. **如果别人抢先做了，你会放弃吗？**
   - 逃避型：会放弃
   - 机会型：可能会放弃
   - 使命型：不会放弃，因为问题还没解决

## 方法论框架

### 动机评估框架

```
动机强度 = 痛苦程度 × 解决欲望 × 持久性

• 痛苦程度：这个问题让你多难受？
• 解决欲望：你多想解决这个问题？
• 持久性：这个欲望能持续多久？
```

### 初心检验清单

| 检验维度 | 逃避型 | 机会型 | 使命型 |
|----------|--------|--------|--------|
| **驱动力** | 逃避痛苦 | 追求机会 | 解决问题 |
| **持久性** | 短期 | 中期 | 长期 |
| **抗挫折** | 弱 | 中等 | 强 |
| **成功概率** | <10% | ~50% | >70% |

## 行动检查清单

### 动机自我检验

- [ ] 能清晰说出想解决什么问题
- [ ] 能描述这个问题为什么重要
- [ ] 能说明为什么是你来解决
- [ ] 能回答"如果失败还愿意做吗"
- [ ] 能描述10年后希望看到什么变化

### 创业准备评估

- [ ] 评估财务准备（至少6个月生活费）
- [ ] 评估家庭支持度
- [ ] 评估风险承受能力
- [ ] 评估退出方案（如果失败怎么办）
- [ ] 评估最佳创业时机

### 初心记录

- [ ] 写下创业初心（一段话）
- [ ] 设定定期回顾机制（每月/每季）
- [ ] 找到初心见证人（分享给朋友）
- [ ] 创建初心提醒物（桌面壁纸/便签）
- [ ] 记录初心动摇时刻及应对方法

## 实用对话模板

### 场景：创业动机评估

**用户**："我想创业，但不知道该不该开始"

**AI（使用本技能）**：
> "让我们用'初心三问'来检验：
> 
> **第一问：你想解决什么问题？**
> 不要说'我想创业'，要说'我想解决[具体问题]'。
> 能告诉我你想解决什么问题吗？
> 
> **第二问：这个问题为什么重要？**
> 对你重要？对社会重要？对用户重要？
> 为什么非得解决？
> 
> **第三问：为什么是你来解决？**
> 你有什么独特优势？
> 为什么别人没做好？
> 
> 回答完这三个问题，我们就能判断你的动机类型了。"

## 参考案例

### 案例1：逃避型动机的失败

**背景**：程序员小李，讨厌996，想创业

**动机**："不想上班，想自由"

**结果**：
- 创业后发现比上班更累（每天工作14小时）
- 没有明确要解决什么问题
- 6个月后资金链断裂，回到职场

**教训**：逃避型动机无法支撑创业长期的艰难。

### 案例2：使命型动机的成功

**背景**：医生小王，发现病历系统效率极低

**动机**："医生应该专注治病，不是填表格"

**结果**：
- 坚持用AI技术解决病历记录问题
- 前3年几乎没有收入
- 第4年产品成熟，获得投资，服务1000+医院

**关键**：使命型动机让他扛过艰难时期。

---

**核心结论**：创业的初心决定了你能走多远。使命型动机 > 机会型动机 > 逃避型动机。
