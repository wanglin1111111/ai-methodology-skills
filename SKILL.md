# 技能包索引

- 功能：营销文案、内容创作辅助
- 文件：`内容创作技能包/SKILL.md`

## 2. 写作辅助技能包
- 位置：`写作辅助技能包/`

## 技能包结构

每个技能包包含以下文件：
- `SKILL.md` - 技能配置文件（必需）
- `README.md` - 使用说明（可选）
- 其他资源文件（可选）

## 使用方法

将技能包目录复制到OpenClaw技能目录：
```bash
cp -r [技能包名称] ~/.openclaw-autoclaw/skills/
```
