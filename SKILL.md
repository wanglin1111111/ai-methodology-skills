# Learning Assessment & Intuition Decision-Making Skill

## 技能描述
管理学习评估与直觉决策知识库，支持评估方法咨询、直觉决策分析、竞争策略咨询、采购优化建议等功能。用于评估方法研究、决策优化、竞争分析等场景。

**隐私保护声明**：
- 本技能仅处理公开的讨论内容
- 不存储、传输或暴露任何个人隐私数据
- 所有知识库内容均为公开可分享的分析观点
- 不涉及用户个人信息、账号数据或敏感业务数据

## 触发条件
- 用户询问评估方法相关问题时
- 用户询问直觉决策时
- 需要竞争策略咨询时
- 需要采购优化建议时
- 用户提到"评估"、"评分"、"直觉"、"竞争"、"采购"、"函数组合"等关键词

## 知识库位置
`knowledge-base/learning-assessment-intuition-decision-making.md`

## 核心功能

### 1. 评估方法咨询
```javascript
// 查询评分选择策略
queryAssessment('评分选择')

// 查询评估态度
queryAssessment('评估态度')

// 查询代理评估
queryAssessment('代理评估')
```

### 2. 直觉决策分析
分析直觉在决策中的作用：
- 直觉的定义与优势
- 直觉 vs 传统方法对比
- 直觉引导的函数构建流程

### 3. 竞争策略咨询
提供竞争层级分析：
- 根本竞争（核心能力）
- 类别竞争（细分领域）
- 次级竞争（具体产品）
- 季节性因素影响

### 4. 采购优化建议
提供采购流程优化方案：
- 采购流程模型
- 钉子户现象分析
- 函数组合优化策略

## 使用示例

### 示例1：评估方法咨询
```
用户：如何选择合适的评分标准？
助手：[读取知识库] 不能仅依赖常规方法，需要结合直觉。具体策略：
1. 识别关键元素
2. 设计组合函数
3. 优化效率
4. 验证迭代
传统方法局限：标准固化、主观性强、效率低下...
```

### 示例2：直觉决策分析
```
用户：直觉在决策中有什么作用？
助手：[读取知识库] 直觉是基于经验和知识的快速、无意识的判断能力。
优势：速度快、灵活性强、创新性高、复杂场景适应性好
应用：直觉引导函数构建，提升组合效率...
```

### 示例3：竞争策略咨询
```
用户：如何理解竞争层级？
助手：[读取知识库] 竞争存在三个层级：
1. 根本竞争：核心能力、资源，关注生存与发展
2. 类别竞争：细分领域、方向，关注定位与差异化
3. 次级竞争：具体产品、服务，关注细节与体验...
```

### 示例4：采购优化建议
```
用户：采购流程中的钉子户现象如何解决？
助手：[读取知识库] 钉子户现象特征：阻力大、影响广、需特别处理。
解决思路：
1. 识别根因：深入分析顽固问题的本质
2. 策略调整：针对不同情况采用不同方法
3. 长期规划：预防类似问题的再次出现...
```

### 示例5：感官与情感分析
```
用户：感官和情感在评估中有什么关系？
助手：[读取知识库] 感官-情感模型：
客观事物 → 感官感知 → 情感赋义 → 主观认知
启示：主观情感是评估过程的组成部分，不可忽视...
```

```markdown
## 学习评估与直觉决策摘要

### 核心观点
1. **评估态度**：放弃心态影响评估效果
2. **情感因素**：感官与情感相互作用
3. **竞争层级**：根本→类别→次级三层
4. **直觉价值**：提升效率的关键

### 直觉优势
- 速度快：即时判断
- 灵活性：适应性强
- 创新性：突破常规
- 复杂场景：游刃有余

### 函数组合策略
元素识别 → 函数设计 → 效率验证 → 持续优化

### 评估方法对比
| 维度 | 传统方法 | 直觉方法 |
|------|---------|---------|
| 速度 | 慢 | 快 |
| 灵活性 | 低 | 高 |
| 创新性 | 弱 | 强 |

### 待验证假设
- [ ] 直觉方法准确率
- [ ] 函数组合标准化
- [ ] 季节性因素量化
```
