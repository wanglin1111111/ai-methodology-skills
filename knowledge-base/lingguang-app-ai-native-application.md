## 产品概况

**灵光（Ling Guang）App** 于去年年底推出，是一款基于代码生成架构的AI原生应用。核心理念是通过自然语言指令，让用户在不到一分钟的时间内生成应用程序（即"Flash Apps"），并支持用户自行修改或与朋友协作remix（混改）。

**关键数据**：
- 平台已有超过3000万个用户创建的Flash Apps
- 演讲者：马奇士（团队代表）
- 推出时间：去年年底

---

## 一、核心架构与技术

### 基于代码生成架构

#### 架构特点
```
自然语言指令 → AI代码生成 → Flash Apps应用 → 用户修改/remix
```

#### 核心优势
- **快速生成**：不到一分钟创建应用
- **低门槛**：无需编程知识
- **可修改**：支持用户自行调整
- **协作性**：可与朋友remix混改

---

## 二、核心功能：Flash Apps（闪应用）⭐

### "许愿即得"（Wish-coding）理念

#### 核心定义
> 通过自然语言指令，快速生成应用程序，实现"许愿即得"

#### 应用场景矩阵

**场景1：日常效率工具**
- **待办事项管理**：个性化任务管理
- **财务规划**：预算跟踪与分析
- **其他工具**：根据需求快速创建

**场景2：互动游戏**
- **游戏开发**：降低开发门槛
- **创意测试**：快速验证游戏创意
- **分享协作**：基于他人创意二次开发

**场景3：3D实时视频**
- **实时内容**：动态3D视频生成
- **互动体验**：沉浸式内容消费
- **创意表达**：多维度内容创作

**场景4：一次性应用** ⭐
- **临时需求**：解决特定临时问题
- **快速响应**：即时生成、即时使用
- **低成本**：无需长期维护

## 四、AI伴侣与视觉交互

### Local Call功能 ⭐

#### 核心定位
- **定位**：亲切、共情的AI伴侣
- **功能**：实时语音对话
- **场景**：
  - casual chat（日常聊天）
  - 咨询问题（问题解答）

#### 设计理念
```
AI伴侣 = 亲切 + 共情 + 实时响应
```

---

### HR Camera / AI Camera功能 ⭐

#### 核心能力矩阵

**能力1：看（识别）**
- **动态场景识别**：实时环境理解
- **野花识别**：提供植物知识解答
- **博物馆导览**：展品讲解与历史介绍

**能力2：聊（交互）**
- **实时对话**：基于视觉信息的语音交流
- **知识补充**：识别后的深度讲解
- **场景适配**：根据场景调整对话内容

**能力3：创造（生成）**
- **拍照生成**：新图片/视频创作
- **艺术灵感**：记录创意想法
- **二次创作**：基于现有素材创新

---

### 应用场景详解

#### 场景1：户外探索
```
看到野花 → AI Camera识别 → 提供植物知识 → 语音讲解 → 深度学习
```

#### 场景2：博物馆参观
```
看到展品 → AI Camera识别 → 讲解展品历史 → 语音导览 → 深度体验
```

#### 场景3：艺术创作
```
拍照 → AI生成新图片/视频 → 记录艺术灵感 → 分享创作 → 社区互动
```

### 核心理念
> 赋予手机"看、聊、创造"的能力
