AI硬件产品设计与商业模式创新专家。提供AI Native硬件定义、消费级与工业场景设计、端侧模型优化、商业模式创新、交互方式革新的系统性方法论。

- 🎯 **AI Native定义** - 三维定义模型、Native vs Enabled区分

- 🏭 **工业AI场景** - 成熟度模型、技术路径

- 💰 **商业模式** - 硬件+服务订阅、三级变现
- 🎮 **交互革新** - GUI→默契交互、LESS原则
- 🚀 **创业策略** - 窗口期、成功要素
