# Test Cases for ai-era-career-transition

## Test Suite: Career Transition Assessment

### Test 1: Basic Transition Inquiry
**Input**: "AI冲击下，设计师如何转型？"

**Expected Output**:
- Acknowledge AI impact on design industry
- Identify transition options: 
  - UX Strategy (high-level)
  - AI Tool Specialist
  - Creative Direction
  - Design Education
- Provide capability gap analysis
- Suggest learning path

**Validation Points**:
- [ ] Multiple transition paths provided
- [ ] Capability assessment included
- [ ] Actionable next steps given

### Test 2: Differentiation Strategy
**Input**: "如何找到与AI的差异化？"

**Expected Output**:
- Explain AI limitations (context, emotion, innovation)
- Identify human strengths
- Provide differentiation framework
- Include self-assessment questions

**Validation Points**:
- [ ] AI vs Human comparison clear
- [ ] Specific differentiation areas identified
- [ ] Framework provided

### Test 3: Capability Building Plan
**Input**: "我需要培养哪些新能力？"

**Expected Output**:
- List AI-era core competencies
- Prioritize based on current role
- Provide learning resources
- Set milestone timeline

**Validation Points**:
- [ ] Competency list relevant
- [ ] Prioritization logic clear
- [ ] Resources actionable

## Test Suite: Industry-Specific Scenarios

### Test 4: Marketing Professional
**Input**: "AI可以写文案、做投放了，营销人怎么办？"

**Expected Output**:
- Acknowledge automation of tactical work
- Identify strategic value: brand strategy, customer insight, creative direction
- Provide upskilling roadmap

### Test 5: Software Developer
**Input**: "AI编程助手这么强，程序员会被取代吗？"

**Expected Output**:
- Clarify AI as tool, not replacement
- Identify new roles: AI solution architect, prompt engineer, system designer
- Provide transition guidance
