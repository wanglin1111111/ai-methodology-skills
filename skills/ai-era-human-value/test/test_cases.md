# Test Cases for ai-era-human-value

## Test Suite: Core Value Identification

### Test 1: Basic Value Discovery
**Input**: "我是一名设计师，AI可以生成图片了，我还有什么价值？"

**Expected Output**:
- Identify user's concern about AI replacing design work
- Explain AI generates images but doesn't understand context
- Highlight value: "需求表达能力" + "审美判断"
- Provide actionable framework for value positioning

**Validation Points**:
- [ ] Response addresses AI concern
- [ ] Identifies at least 2 human-unique values
- [ ] Provides concrete next steps

### Test 2: Scarcity Analysis
**Input**: "分析短视频创作者在AI时代的稀缺性"

**Expected Output**:
- Analyze current market: AI tools lower video production barrier
- Identify new scarcity: "独特视角" + "真实体验" + "情感连接"
- Provide positioning framework

**Validation Points**:
- [ ] Market analysis included
- [ ] New scarcity identified
- [ ] Framework provided

### Test 3: Demand Expression Training
**Input**: "如何培养我的需求表达能力？"

**Expected Output**:
- Define demand expression: clarity + specificity + context
- Provide 3-step training method
- Include practice exercises

**Validation Points**:
- [ ] Clear definition provided
- [ ] Step-by-step method given
- [ ] Practice exercises included
