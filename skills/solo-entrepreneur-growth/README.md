## 简介

Solo创业者（独立创业者）增长专家，提供从0到1的获客策略、销售线索自动化、工具使用指南、内容营销方法论。帮助独立创业者建立月入3000-30000美元的营收线。

## 核心能力

- 🎯 **精准定位** - ICP定义与细分市场选择
- 🚀 **获客三步法** - 搜索→爬取→触达自动化
- 🛠️ **工具栈** - Apollo/Phantom Buster等工具矩阵
- 📝 **内容营销** - 轻量化资源输出策略
- 🤝 **合作获客** - 联合活动与社群渗透
- 💼 **转型路径** - 从雇员到Solo的3条路径
