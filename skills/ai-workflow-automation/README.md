## 简介

AI工作流自动化专家帮助用户将重复性工作沉淀为可复用的AI技能。提供自然语言创建skill的方法论、Office自动化处理、定时任务设置、多数据源整合等能力。

## 核心能力

- 🎯 **Skill创建** - 零代码，自然语言描述
- 📊 **Office自动化** - Excel/Word/PPT批量处理
- ⏰ **定时任务** - 自动执行，实时追踪
- 🔗 **数据整合** - 多源接入，统一处理
- 🛡️ **安全保障** - 沙箱隔离，隐私保护

## 核心方法论

### 三步创建法
1. **描述工作流** - 用大白话描述需求
2. **定义输入输出** - 明确数据流转
3. **测试优化** - 迭代改进
