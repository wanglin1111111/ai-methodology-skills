# Test Cases for self-as-product-methodology

## Test Suite: Three-Step Framework

### Test 1: Uniqueness Assessment
**Input**: "我有什么独特之处？"

**Expected Output**:
- Guide user through uniqueness discovery
- Questions about: experiences, perspectives, combinations
- Uniqueness formula: rare combination + personal story
- Actionable positioning statement

**Validation Points**:
- [ ] Discovery process structured
- [ ] Uniqueness formula applied
- [ ] Positioning statement generated

### Test 2: Life Parameters Analysis
**Input**: "如何丰富我的生命参数？"

**Expected Output**:
- Define life parameters concept
- Assess current parameters
- Identify expansion opportunities
- Provide enrichment roadmap

**Validation Points**:
- [ ] Concept clearly explained
- [ ] Assessment practical
- [ ] Roadmap actionable

### Test 3: Charisma Building Plan
**Input**: "如何构建我的个人魅力？"

**Expected Output**:
- Define charisma in product terms
- Identify charisma components
- Provide building exercises
- Set measurement criteria

**Validation Points**:
- [ ] Definition clear
- [ ] Components actionable
- [ ] Exercises practical
