## 核心方法论

### 创作流程四步法
1. **需求描述** - 用大白话描述想法
2. **功能拆解** - 列出核心功能点
3. **视觉定义** - 描述风格要求
4. **迭代优化** - 测试反馈调整

### Agent设计三原则
1. **工具优先论** - 工具体系70%，AI处理30%
2. **边界明确** - 清晰定义能力范围
3. **可靠性优先** - 关键节点人工确认
