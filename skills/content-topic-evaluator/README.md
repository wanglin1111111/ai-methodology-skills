# 内容选题评估器 (Content Topic Evaluator)

[![Version](https://img.shields.io/badge/version-1.0.0-blue)]()
[![Stability](https://img.shields.io/badge/stability-stable-green)]()

> 基于HKR三维框架，系统化分析内容选题质量，帮助创作者做出数据驱动的选题决策。

## 核心能力

- 📊 **HKR三维评估** - Happy趣味性、Knowledge信息量、Resonance情绪共鸣
- 🎯 **分级定级** - S/A/B/C/D五级评级系统
- 📈 **批量评估** - 支持多选题批量评分和优先级排序
- 💡 **优化建议** - 针对弱项提供具体改进方向

## HKR评估框架

| 维度 | 评估内容 | 权重 |
|------|---------|------|
| **H - Happy** | 趣味性/吸引力 | 33% |
| **K - Knowledge** | 信息量/价值感 | 33% |
| **R - Resonance** | 情绪共鸣/代入感 | 33% |

## 评级标准

| 总分 | 评级 | 建议 |
|------|------|------|
| 13-15分 | S级 | 爆款潜力，优先制作 |
| 10-12分 | A级 | 优质内容，正常推进 |
| 7-9分 | B级 | 及格内容，需优化 |
| 4-6分 | C级 | 建议淘汰或重选 |
| 3分以下 | D级 | 直接放弃 |

## 适用场景

- 自媒体文章选题
- 短视频脚本评估
- 播客主题筛选
- 公众号推文规划
- 直播话题选择

## 使用方法

1. 提供内容主题和目标受众
2. 获取HKR三维评分
3. 查看综合评级和优化建议
4. 批量评估时获取优先级排序

## 核心公式

```
内容质量 = H(趣味性) × K(信息量) × R(共鸣感)

爆款潜力 = 高分选题 × 优质制作 × 精准分发
```
